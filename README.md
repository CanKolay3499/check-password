# c-template

## Installation

#### Requirements

1. Clang (gcc)
2. CMake

#### Clone the repo

```sh
git clone https://github.com/CanKolay3499/c-template && cd c-template/
```

#### Compile the code

```sh
mkdir -p build/ && cd build/
cmake ../ && make -j$(nproc)
```

if you are on the fish shell remove the `$` sign.

#### Execute the `out`

```sh
./out
```
